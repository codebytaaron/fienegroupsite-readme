## About Fiene Group
This is a showcase of the **Fiene Group public site** — a modern, high-performance website designed to highlight the brand, vision, and community of Fiene Group.  

The **full source code is private**, but this repository demonstrates the project structure, features, and technologies used.

---

## Key Features
- **Modern Frontend:** Built with **React + Vite** for fast performance and modular development.  
- **Responsive Design:** Fully responsive with **TailwindCSS**, including dark/light theme toggle.  
- **Animations:** Smooth scroll and hover animations using **Framer Motion**.  
- **Modular Components:** Hero section, About, Vision, Team, Footer — easily extendable.  
- **Performance Focused:** Optimized build with Vite for speed and minimal load times.

---

## Tech Stack
- **Frontend:** React, Vite  
- **Styling:** TailwindCSS (dark/light theme ready)  
- **Animations:** Framer Motion  
- **Build & Deployment:** Node.js, npm scripts
